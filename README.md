<h2>CatShell</h2>
 игрушечный учебный проект с дебильным парсером команд на else if.

 юзается экспериментальная либа #include <experimental/random> поэтому осторожнее.
  
 список команд смотреть в самой программе.


<h2>Компиляция</h2>
- clang: clang++ -std=c++17 -fexperimental-library CatOS.cpp -o catshell
<p>
- gcc: g++ CatOS.cpp -o catshell (ничего необычного)
